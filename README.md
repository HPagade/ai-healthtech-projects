## 📂 Projects

### 1. 🚀 YC AI Healthtech Startup Tracker
*Analyzing 200+ AI and healthcare startups to identify market trends and funding patterns*
- **Tech Stack:** Python, BeautifulSoup, Pandas, Matplotlib
- **Status:** ✅ Complete
- **Folder:** `01-yc-healthtech-tracker/`
- **Features:** Web scraping, data analysis, trend visualization

### 2. 📊 Customer Health Score Calculator
*Predicting customer churn using usage metrics with interactive dashboard*
- **Tech Stack:** Python, Streamlit, Pandas, Plotly
- **Status:** ✅ Complete
- **Folder:** `02-customer-health-score/`
- **Features:** Interactive dashboard, churn prediction, customer analytics

### 3. 🤖 AI Cover Letter Generator
*Automating personalized cover letter creation for startup applications*
- **Tech Stack:** Python, OpenAI API, CLI
- **Status:** ✅ Complete
- **Folder:** `03-ai-cover-letter/`
- **Features:** GPT-4 powered, interactive CLI, profile saving

### 4. 🏥 AI Clinical Decision Support Tool
*Prototype symptom checker using decision trees and ML*
- **Tech Stack:** Python, Scikit-learn, Random Forest
- **Status:** ✅ Complete
- **Folder:** `04-clinical-decision-support/`
- **Features:** ML-based diagnosis, interactive symptom checker, clinical recommendations

### 5. 📈 Startup Job Market Analysis
*Analyzing job postings from YC/healthtech startups to identify hiring trends*
- **Tech Stack:** Python, Web Scraping, Data Analysis
- **Status:** ✅ Complete
- **Folder:** `05-job-market-analysis/`
- **Features:** Job scraping, salary analysis, skills demand tracking

### 6. 💡 AI Healthtech Product Teardown
*Deep strategic analysis framework for leading AI healthtech products*
- **Tech Stack:** Research, Analysis, Strategic Thinking
- **Status:** ✅ Complete
- **Folder:** `06-product-teardown/`
- **Features:** Analysis templates, example teardowns, competitive frameworks

### 7. 💰 Healthcare Startup Funding Analysis
*Interactive dashboard analyzing AI healthtech funding trends*
- **Tech Stack:** SQL, Python, Tableau Public
- **Status:** ✅ Complete
- **Folder:** `07-funding-analysis/`
- **Features:** SQL queries, Python analysis, Tableau integration, trend analysis

### 8. 🤖 AI Patient Triage Chatbot
*Conversational AI agent for symptom assessment and triage*
- **Tech Stack:** Python, LangChain, Streamlit, OpenAI GPT-4
- **Status:** ✅ Complete

- **Features:** AI conversation, triage recommendations, web interface
